# 🏠 facebook-apartments

כלי אישי לחיפוש דירות — סורק פוסטים מקבוצות פייסבוק, מדלן ויד2, מאחסן הכל ב-SQLite, ומציג ממשק Streamlit עם מפה, פילטרים, ומעקב אישי.

---

## מה זה עושה

- **סורק** קבוצות פייסבוק, מדלן ויד2 לפי הגדרות מחיר/חדרים/גודל
- **שומר** כל מודעה ב-SQLite — כולל היסטוריית שינויי מחיר
- **מציג** ממשק ווב בעברית (RTL) עם טבלה, מפה ופרטים מורחבים
- **עוקב** אחרי מה שראית, כמה ימים דירה בשוק, וירידות מחיר

---

## דרישות מקדימות

- Python 3.10+
- חשבון פייסבוק עם גישה לקבוצות הרלוונטיות

## התקנה

```bash
# 1. שכפל את הריפו

# 2. התקן תלויות
```

```yaml
  כל_כמה_שעות: 2
```

## התחברות לפייסבוק (פעם אחת)

הסריקה של פייסבוק דורשת session שמור. להתחבר:

```bash
python save_session.py
```

ייפתח דפדפן — התחבר לפייסבוק ידנית, ואז הסקריפט ישמור את ה-session ל-`fb_session.json`.

> **שים לב:** `fb_session.json` מכיל את פרטי ההתחברות שלך — לעולם אל תעלה אותו ל-Git (כבר מוגדר ב-`.gitignore`).

## פיצ׳רים בממשק

### טבלת דירות
| סמל | משמעות |
|-----|--------|
| 🆕 | נמצאה היום |
| 📉 | מחיר ירד מאז הסריקה הראשונה |
| 👁 | סימנת כ"ראיתי" |
| 🚫 | סימנת כ"הוסרה" |

### פרטי דירה
- **ימים בשוק** — מחושב מתאריך הפרסום המקורי. ירוק = חדש, כתום = מחכה, אדום = כוח מיקוח גבוה
- **ירידת מחיר** — אם המחיר השתנה בין סריקות, מציג כמה ירד ובכמה אחוז
- **ראיתי / בטל** — מסמן את הדירה כנבדקה, ניתן לסנן "הסתר ראויות"
- **סמן כהוסרה** — להסתיר מודעות לא רלוונטיות
- **הערות** — שדה חופשי לכל דירה, נשמר ב-DB

### מפה
מציגה את כל הדירות עם קואורדינטות. לחיצה על סמן פותחת פרטים. סמנים ירוקים = דירות שראית.

## הערות על מדלן

מדלן משתמשת ב-PerimeterX להגנה מסריקה. יש שתי גישות:

1. **CDP** — הפעל `python start_madlan_browser.py` פעם אחת, ואז הסריקה מתחברת לדפדפן הפתוח
2. **פרופיל** — הפעל `python save_madlan_session.py` לשמירת session

בפועל הסריקה מנסה CDP קודם, ואם לא זמין — עוברת לפרופיל שמור.
